Projects

Audio Analyzer

This project was a hands-on electronics and signal processing build where I designed a hardware-based audio analyzer. The system processes audio signals through active filters and uses LEDs to visualize frequency bands in real time. It showcases both analog circuit design and practical embedded systems implementation.

Other Projects

  • ToyG Compiler: Created a compiler in C using Lex and Yacc to generate x86-64 assembly from a simplified toy language.
  • Postfix Calculator: Implemented a reverse Polish notation calculator in C using a custom stack built on a doubly-linked list.
  • Linked List Library: Developed a circular doubly-linked list library in C with dynamic memory handling and pointer logic.
  • Multisim Circuit Labs: Designed and simulated various analog circuits, including filters, op-amps, and MOSFET-based amplifiers.

Currently Working On

I’m currently expanding the Audio Analyzer project to support a greater number of frequency bands. This upgraded version will use finely tuned filters and multiple LED columns to detect specific frequency regions like bass, midrange, and treble within a song — creating a more immersive and informative audio visualization experience.
